# 📄 PDF Analyzer & AI Research Assistant

사용자가 업로드한 PDF 파일을 브라우저에서 즉시 확인하고, **OpenAI GPT-4o-mini**를 활용하여 내용을 자동으로 요약 및 분석해 주는 웹 애플리케이션입니다.

---

## 🚀 주요 기능
* **PDF 실시간 뷰어**: 업로드한 PDF를 브라우저 왼쪽 화면에서 바로 확인 가능
* **AI 자동 요약**: PDF 내 텍스트를 추출하여 GPT가 핵심 내용을 한국어로 요약
* **기술 용어 분석**: 텍스트 분석 API를 통해 복잡한 전문 용어 해설 지원
* **반응형 레이아웃**: 뷰어와 분석창이 분리된 2단 구성의 인터페이스

---

## 🛠 기술 스택

### **Backend**
* **FastAPI**: 고성능 비동기 웹 프레임워크
* **PyMuPDF (fitz)**: PDF 텍스트 추출 및 처리
* **OpenAI SDK (v1.0+)**: GPT-4o-mini 모델 연동
* **Python-dotenv**: 환경 변수 및 API Key 관리

### **Frontend**
* **PDF.js**: 클라이언트 측 PDF 렌더링 라이브러리
* **Vanilla JS**: Fetch API를 활용한 서버 통신 및 DOM 조작

---

## 🏗 시스템 구조 (System Architecture)

작성된 시스템은 다음과 같은 흐름으로 작동합니다.

| 단계 | 구성 요소 | 역할 |
| :--- | :--- | :--- |
| **1. 파일 선택** | HTML (`input`) | 사용자가 로컬 PC에서 PDF 파일을 선택합니다. |
| **2. 화면 렌더링** | JS (`embed`) | 브라우저 왼쪽 영역에 PDF 미리보기를 생성합니다. |
| **3. 데이터 전송** | JS (`Fetch API`) | `FormData`에 파일을 담아 FastAPI 서버로 전송합니다. |
| **4. 서버 처리** | FastAPI (`Python`) | 파일을 저장한 뒤, **PyMuPDF**로 텍스트를 추출하여 **GPT**에 전달합니다. |
| **5. 결과 출력** | Sidebar (`div`) | GPT가 분석한 한국어 요약본을 오른쪽 사이드바에 표시합니다. |

---

## 💡 개선 예정 사항 (UX Enhancements)

* **로딩 스피너 도입**: GPT 분석 시간(약 5~10초) 동안 사용자에게 시각적 피드백을 제공하기 위해 CSS 스피너 또는 로딩 애니메이션을 추가할 예정입니다.
* **텍스트 드래그 분석 (Selective Analysis)**: PDF 전체 요약뿐만 아니라, 사용자가 뷰어에서 마우스로 드래그한 특정 문장만 선택하여 번역하거나 해설하는 기능을 추가할 예정입니다.
* **파일 보안**: 업로드된 파일의 확장자 검증 및 용량 제한 로직을 강화하여 서버 안전성을 확보합니다.
